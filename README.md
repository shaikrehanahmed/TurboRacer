# 🏎️ Turbo Racer: Ultimate Edition

A high-speed, 2D top-down racing game built from scratch without any game engines. 

This project was built to push the limits of pure DOM manipulation, Vanilla JavaScript game loops, and advanced CSS rendering (including generating realistic 3D-looking vehicles and textures entirely out of CSS gradients and shadows).

## ✨ Features

* **Dynamic Environments:** Switch seamlessly between Forest, Desert, Snow, and Nightmare biomes.
* **Pure CSS Graphics:** No external image assets were used. The cars, road textures, and obstacles are rendered 100% via CSS.
* **Custom Game Engine:** Built with a custom `requestAnimationFrame` loop handling physics, speed scaling, and bounding-box collision detection.
* **Multi-User Authentication:** A built-in login system utilizing `localStorage` to save individual player states, high scores, coin balances, and inventory.
* **Weapon Garage:** Collect coins during the race to buy and equip different weapons (Blasters, Lasers, Plasma) to destroy obstacles.

## 🛠️ Tech Stack

* **HTML5:** Structure and UI overlays (Glassmorphism login panels).
* **CSS3:** Advanced styling, animations, radial/linear gradients, and SVG noise textures.
* **Vanilla JavaScript (ES6+):** Game logic, state management, audio synthesis, and DOM updates.

## 🚀 How to Play

Since this is a client-side web game, no installation or server is required!

1. Clone the repository or download the `.zip`.
2. Open `index.html` in any modern web browser.
3. Create an account on the login screen.
4. Use the **Left/Right Arrows** or **A/D** keys to steer.
5. Press **Spacebar** to shoot your equipped weapon.
6. Survive, collect coins, and beat the high score!
